To define a process, its name must be specified after the reserved word [PROCESS](declaration_of_processesdot.md).

### Example
```
PROGRAM my_game;
BEGIN
    //...
END

PROCESS my_process()
BEGIN
    //...
END
```


This process has been declared with the name **my_process**.
